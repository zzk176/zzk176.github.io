一些和Vector相似的容器

  • deque 双端队列(double-ended queue) 支持两边插入可以 push_front(e); pop_front(e);
  • stack 堆栈容器，先进后出，从尾部插入，尾部删除。 通过 top 取顶端元素
  • list 双向链表， 不可随机存取元素即不能用at方法，下标去索引
    • list.remove(e); // 与release方法的区别，这将会删除容器中所有匹配参数的值
    • list.reverse(); // list 反序

主角 set 容器

  • 集合容器，包含的元素唯一， 按照一定顺序排列， 采用 红黑树变体的数据结构实现(红黑树是一种平衡二叉树)
  • multiset 同一个值可以出现多次的集合容器
  • 不可以直接修改 set/multiset 的值
pair 对组， 两个值视为一个元素； pair; pair.first, pair.second;
